# Technical Evaluation Rubric

The 12-question evaluation uses a 0-2 scoring scale.

## General scoring scale

- 0 = Criterion not met
- 1 = Criterion partially met
- 2 = Criterion fully met
- N/A = Criterion does not apply to this question

## Retrieval relevance

- 0: Retrieved sources are unrelated to the question.
- 1: At least one source is partially relevant.
- 2: The most important retrieved sources directly match the expected topic.

## Answer faithfulness

- 0: The answer contains unsupported or contradictory claims.
- 1: The main answer is supported, but some details are unclear or unsupported.
- 2: The answer is fully supported by the retrieved source evidence.

## Citation accuracy

- 0: Citations are missing or do not support the associated claims.
- 1: Some citations are correct, but citation coverage is incomplete.
- 2: Citations are present and accurately support the answer claims.

## Novice readability

- 0: The answer is confusing or unsuitable for a beginner.
- 1: The answer is understandable but could be clearer or simpler.
- 2: The answer is clear, concise, and beginner-friendly.

## Refusal appropriateness

Use for unsupported or out-of-scope questions.

- 0: The system guesses or provides an unsupported answer.
- 1: The system expresses uncertainty but still includes unsupported content.
- 2: The system appropriately refuses or redirects without unsupported claims.

## Staged hint quality

Use primarily for Q007.

- 0: The system immediately provides the complete solution.
- 1: The system provides a partial hint but reveals too much.
- 2: The system provides an appropriately staged hint without immediately giving the full answer.

## Overall pass

Mark Yes when the system meets the main expected behaviour for the question.

Mark No when the main expected behaviour is not achieved.
