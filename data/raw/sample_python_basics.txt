Python Basics Sample Source

A variable is a name that stores a value. In Python, a variable is created when a value is assigned to it using the assignment operator. For example, x = 5 creates a variable called x and stores the value 5.

A string is a sequence of characters. Strings in Python are usually written inside quotation marks. For example, name = "Ana" stores the text Ana in a variable called name.

A list is a collection of values. Python lists use zero-based indexing, which means the first item is at position 0, not position 1.

A for loop is used to repeat code for each item in a sequence. For example, for number in numbers repeats the loop body once for each value in the list.

A while loop repeats code while a condition is true. A while loop is useful when the number of repetitions is not known in advance.

An if statement is used to make decisions in a program. In Python, an if statement must end with a colon. For example, if x > 3: is valid syntax.

A function is a reusable block of code. Functions help programmers organise code, avoid repetition, and make programs easier to understand.

An accumulator pattern is often used to calculate a total. A variable such as total is first set to 0, and then each number is added to the total inside a loop.
