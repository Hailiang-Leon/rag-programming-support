Python Lists

A list is a data structure that stores multiple values in a single variable. Lists are one of the most commonly used data structures in Python.

Creating a List

You create a list by placing values inside square brackets, separated by commas.

    fruits = ["apple", "banana", "cherry"]
    numbers = [1, 2, 3, 4, 5]
    mixed = [10, "hello", True]

Lists can store any type of value, and they can even mix different types, though this is not always recommended.

An empty list is created with empty square brackets.

    empty = []

Accessing Items by Index

Each item in a list has a position called an index. In Python, indexing starts at 0, not 1. This is called zero-based indexing.

    fruits = ["apple", "banana", "cherry"]
    print(fruits[0])    # apple
    print(fruits[1])    # banana
    print(fruits[2])    # cherry

The first item is at index 0, the second at index 1, and the third at index 2.

This is one of the most common misconceptions for beginners. Python does not start counting from 1. It always starts from 0.

Negative Indexing

You can also access items from the end of the list using negative indices.

    fruits = ["apple", "banana", "cherry"]
    print(fruits[-1])    # cherry
    print(fruits[-2])    # banana

Index -1 refers to the last item, -2 to the second-to-last item, and so on.

IndexError

If you try to access an index that does not exist, Python will raise an IndexError.

    fruits = ["apple", "banana", "cherry"]
    print(fruits[5])    # IndexError: list index out of range

The list has only three items at indices 0, 1, and 2. Trying to access index 5 causes an error because no item exists there.

Changing an Item

You can change the value at a specific index by assigning a new value to it.

    fruits = ["apple", "banana", "cherry"]
    fruits[1] = "mango"
    print(fruits)

This will output:
    ['apple', 'mango', 'cherry']

Finding the Length of a List

Use len() to find how many items are in a list.

    fruits = ["apple", "banana", "cherry"]
    print(len(fruits))

This will output:
    3

Adding Items

Use append() to add an item to the end of a list.

    fruits = ["apple", "banana"]
    fruits.append("cherry")
    print(fruits)

This will output:
    ['apple', 'banana', 'cherry']

Removing Items

Use remove() to delete a specific value from a list.

    fruits = ["apple", "banana", "cherry"]
    fruits.remove("banana")
    print(fruits)

This will output:
    ['apple', 'cherry']

Looping Through a List

You can use a for loop to go through each item in a list.

    fruits = ["apple", "banana", "cherry"]
    for fruit in fruits:
        print(fruit)

This will output:
    apple
    banana
    cherry

List Slicing

You can access a portion of a list using slicing.

    numbers = [10, 20, 30, 40, 50]
    print(numbers[1:3])

This will output:
    [20, 30]

Slicing gives you the items from the start index up to but not including the end index.

Common Mistakes

Accessing index 0 when you expect position 1 is very common. Always remember that Python lists start at index 0.

Trying to access an index that is equal to the length of the list will cause an IndexError, because the last valid index is always length minus 1.

    numbers = [10, 20, 30]
    print(numbers[3])    # IndexError: valid indices are 0, 1, 2
